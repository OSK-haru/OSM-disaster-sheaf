# TASK_PHASE_5.md

Read `AGENTS.md` and `PROJECT_SCOPE.md` first.

## Phase 5 Goal
Explain why origins are unreachable.

## Required explanation categories
Provide at least 3 categories, such as:
- flood blockage
- stairs/inaccessible segment
- disconnected network or no shelter path

Exact labels may vary, but at least 3 categories must exist.

## Tasks
1. Assign failure reasons to unreachable origins.
2. Export failure reason GeoJSON.
3. Save one failure reason map.
4. Summarize category counts.

## Expected outputs
- failure_reason.geojson
- failure_reason_map.png
- summary table or markdown snippet

## Acceptance criteria
- unreachable outputs are interpretable
- at least 3 categories are present
- failure reasons can be understood without reading code

## Stop after
Failure explanation layer is complete.
Do not implement multiple interventions yet.
