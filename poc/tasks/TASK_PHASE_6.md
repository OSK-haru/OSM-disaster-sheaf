# TASK_PHASE_6.md

Read `AGENTS.md` and `PROJECT_SCOPE.md` first.

## Phase 6 Goal
Run one simple intervention scenario and compare it against the baseline.

## Allowed intervention
Choose exactly one simple intervention, for example:
- add one alternative shelter
- unblock one selected edge
- remove one blocking assumption in a hypothetical scenario

## Tasks
1. Define one intervention scenario.
2. Recompute reachability.
3. Compare baseline and intervention.
4. Export comparison outputs in GeoJSON.
5. Save one comparison figure.

## Expected outputs
- intervention_comparison.geojson
- intervention_comparison.png
- concise comparison summary

## Acceptance criteria
- one measurable difference is shown
- the intervention is clearly documented as hypothetical if needed
- comparison is easy to explain

## Stop after
One intervention comparison is complete.
Do not add multiple intervention scenarios in the first pass.
